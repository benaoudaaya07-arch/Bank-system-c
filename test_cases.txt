
# 🧪 Test Cases – Bank Management System

---

## ✅ Test Case 1: Create a New Account (Valid)

**Input:**

* User selects "Create Account"
* Name: John Doe
* Initial Deposit: 1500
* PIN: 1234

**Expected Output:**

* Account created successfully
* Account ID: 1001
* Balance: 1500.00 DZD

---

## 🔴 Test Case 2: Create Account (Invalid Deposit)

**Input:**

**Expected Output:**

* Error: Initial deposit must be at least 1000 DZD

* Balance: 1200.00 DZD

## 🔄 Test Case 9: Transfer Money (Valid)

**Input:**

* User selects "Create Account"

* Name: Alice

* Deposit: 2000

* PIN: 2222

* User selects "Create Account"

* Name: Bob

* Deposit: 1000

* PIN: 3333

* User selects "Transfer"

* From Account ID: 1002

* PIN: 2222

* To Account ID: 1003

* Amount: 500

**Expected Output:**

* Transfer successful
* New balance (1002): 1500.00 DZD
* New balance (1003): 1500.00 DZD
